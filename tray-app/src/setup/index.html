Obsidian Meeting Notes
Let's get you set up in a few quick steps.
1. Backend API Key
Enter the API key from your registration email.
2. Google Calendar
Connect your Google Calendar so the bot knows when to join meetings.
A browser window will open. Come back here after connecting.
3. LLM Provider
Choose an LLM provider and enter your API key for summarization.
4. Obsidian
Verify the Obsidian CLI is available and optionally set a vault.
